# Ai_Chatbot (PA)

This project features a chatbot application built using [Gradio](https://www.gradio.app/) and OpenAI's GPT-3.5-turbo model. The chatbot allows users to engage in conversations with an AI assistant through a web-based interface.

## Highlights

- **Seamless Interface:** Utilizes Gradio for a clean and responsive user experience.
- **AI-Powered:** Leverages OpenAI's GPT-3.5-turbo for intelligent and context-aware responses.
- **Conversation Memory:** Maintains conversation history for ongoing interactions.

## Prerequisites

- Python version 3.7 or later
- An OpenAI API key


- Configure your OpenAI API key:

   Open the script and replace `"Enter your API Key here"` with your actual OpenAI API key.
